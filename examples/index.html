MorphDB example apps

One generic backend. Five products. Only the schema differs.

Each app is a single HTML file that ships its own morphdb.schema.json and talks to the same generic /objects endpoints — here, a cloud-hosted MorphDB. It registers its app and seeds sample data on first load.
